🔑 Google Maps APIキーの設定

このGoogle版は、地図表示と場所検索に Google Maps Platform を使用します（ルート計算は無料のValhalla公開サーバー）。お手持ちのAPIキーを設定してください。

  1. Google Cloud Console でプロジェクトを作成
  2. 課金を有効化（毎月の無料利用枠あり）
  3. 「Maps JavaScript API」「Places API (New)」「Geocoding API」を有効化
  4. 認証情報から APIキー を作成
  5. キーに「HTTPリファラー制限（ウェブサイト制限）」と「API制限（上記3つのAPIのみ）」を設定

キーはこの端末の localStorage にのみ保存されます（Google APIの呼び出し以外に送信されません）。場所検索はセッショントークン＋Essentialsフィールドのみで課金を最小化しています。
